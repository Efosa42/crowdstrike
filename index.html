Open protocol for connecting Wallets to Dapps

What is WalletProtocols?

WalletProtocols is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making WalletConnect wallets a safer choice compared to desktop or browser extension wallets.

How does it work?

WalletConnect protocols connects web applications to supported mobile wallets WalletConnect Protocol session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).

Faq
  • How do I install WalletProtocol?
    WalletProtocol is not an app, but a protocol supported by many different decentralised applications and wallets. Install any of mobile wallets supporting WalletConnect protocol. WalletConnect wallets are available for Android and iPhone.

  • Is there a token?
    There is no token. WalletProtocol runs on a blockchain and there are no fees.
